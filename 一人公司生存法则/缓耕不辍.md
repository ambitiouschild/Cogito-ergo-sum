# 一人公司，把自己当成一家公司来经营


```中文
	一人公司，把自己当成一家公司来经营。大环境越不好，一人公司越值钱。
	
	战略部
	1.核心资源(Core resurce)：
	我是谁？
	我有什么？
	我如何判断？
	
	技术部
	2.效率工具(Efficiency tools):
	我能卖什么？
	我能利用什么？
	
	品牌部
	3.IP杠杆(IP leverage):
	我如何帮助他人？
	我如何宣传自己？
	
	销售部
	4.开始变现(Start monetizing):
	我的定价？
	盈利模式？
	如何拓展目标用户？
	
	财法部
	5.学会算账(accounting):
	我的收入情况？
	我的成本结构？

	人力部：
	6.社交管理(Social management):
	我如何借势他人？
	我如何管理自己？
	
	团队人数<3人，启动资金<5w，工作时长<8小时
	
	1.搞钱模式，越简单越挣钱
	精简自己的商业模式。
	意识到这点，你就不会沉迷高大上的搞钱逻辑，而是越来越清晰自己的变现方式。
	
	2.记住，没钱是你最大的优势
	有钱 = 被资本绑架，最后变成别人的打工人
	没钱 = 只能靠客户活下来，反而活得更久
	
	3.执行力才是关键
	再牛的点子 x 差劲的执行 = 一文不值
	普通的点子 x 超强的执行 = 持久的价值
	以后别人再说：我有个特别厉害的创业想法！你只需要一笑而过。
	
	4.长远计划没有什么用
	客户需要什么，你就做什么，五年计划等你成为战略决策者再说
	
	5.有趣超级重要
	在搞钱路上，你可以试着开始以有趣服人。
	用情绪价值，真心跟用户做朋友，而不是把他们当做陌生人。
	
	6.成为创造者
	放下着急搞钱心态，沉浸式去工作，享受过程，而不是收款的结果。
	
	7.打造一个乌托邦
	你可以把公司变成自己的乌托邦。意识到创业只是养活自己，你完全可以按照自己的规则，创造一个让自己快乐的事业。
	
	8.成为你想成为的人
	人生真的太短，专注于自己非常想要的目标，不要瞎卷，爱上创业的价值
	
	
	
    
```
