# 拥有生产资料才会财富自由


```中文
    Cogito-ergo-sum：我思故我在，笛卡尔的哲学命题。 意思是思想可以怀疑外在对象，也可以怀疑思想之内的对象，却不能怀疑自身。思想可以怀疑思想的对象和内容，但是不能怀疑“我在怀疑”，否则怀疑活动就无法进行。

    想发财，先辞职。因为上班确实很耽误赚钱。

    一个人如果想要赚到钱，真正的那种利润，而不是所谓的温饱月薪。

    那你一定要拥有一个自己的小规模生意或者说业务。先把自己的大脑进化成一个生意人该有的水平，这才是你发财的第一步。

    而打工或者说上班，它的本质是拒绝你发展，拒绝你进步，逼着你把成长的可能性放弃，只通过提供单一的要素在别人的体系里面混口饭吃。
    
    你可以想想看，我们在上班的这个过程中，是不是都是靠单一的付出来获取报酬的？要么机械的重复做一件事，要么完成一个所谓的任务清单，要么围着一个指标团团转。
    
    所谓的到点下班，更是这种机械化付出的极致体现，因为允许你到点下班的公司，说白了你为此的付出仅仅知识每天8个小时，说的难听点叫出卖八个小时，浪费八个小时，做不做事的不是最重要的，把时间给熬满才是最重要的。
    
    这样的付出久而久之会把人异化，会让人只懂得靠单一的付出去拿钱。如果是这样，你就越来越离不开你的公司，越来越离不开你的那个组织，因为只有那个组织才愿意为你这个单一付出支付报酬。
```
